# References · L2 对话行为模式

**2 条**，覆盖跨轮对话中暴露的认知/行为循环模式。

**状态**：v0.1 骨架——定义和信号需 research 精修，owner 审 tone。

**关键**：L2 触发阈值比 L1 严格得多。模式真实存在，但误触发代价高（用户会感觉被评价）。Aside靠 Claude 的**语义判断**分辨"深入探索" vs "焦虑绕圈"——这是 L2 的核心裁判机制。

---

## 1. 反复确认 / Reassurance-seeking

- **范围**：对话级行为（多轮）
- **signature**（Claude 必须同时验证三条才能触发）：
  1. 过去 5-7 轮内，同一本质问题被用户 rephrase 3+ 次
  2. 每次 rephrase **不加入新信息**——就是同一问题换说法
  3. 用户不接受肯定答复为终点，继续换措辞要更强的确定
- **典型对话**：
  ```
  User:  "这个方案可行吗？"
  Claude: "可行，理由 X Y Z"
  User:  "但如果 A 呢？"
  Claude: "A 不是问题"
  User:  "那 B 呢？也没事吧？"
  Claude: "也没事"
  User:  "你确定吗？完全没问题？"  ← 触发于此
  ```
- **定义**：用户重复询问同一问题或寻求情感确认，以缓解焦虑，不因得到答案而满足。
- **来源**：Abramowitz, Schwartz & Whiteside, 2003（OCD 研究）

### 不是反复确认（看起来像但其实不是）

- **深入探索**：每次 rephrase 加入新信息或试新角度
- **澄清提问**：用户确实没理解之前的答复
- **Power user 思考外化**：多角度代表真实的智力探索，不是焦虑

**Claude 必须区分"有新信息加入 vs 原地绕"才能触发**。不确定时，**不触发**。

## 2. 思维反刍 / Rumination

- **范围**：对话级行为（多轮）
- **signature**（Claude 必须同时验证三条才能触发）：
  1. 用户在同一负面主题上循环 4+ 轮
  2. 用户拒绝了所有提出的方案或视角转换（Claude 的或自己想的都不接受）
  3. 循环**没有推进**——没有新角度、没有决策、纯粹重访问题
- **典型对话**：
  ```
  User:  "为什么我总是搞砸……"
  Claude: [给出一个角度]
  User:  "是的但这次不一样……"
  Claude: [给出另一个角度]
  User:  "都对但偏偏……"
  Claude: [给出行动建议]
  User:  "但我真的不知道为什么……"  ← 原地打转
  ```
- **定义**：用户陷入对同一负面事件或情境的重复思考循环，无法跳出问题框架，拒绝接受解决方案或视角转换。
- **来源**：Nolen-Hoeksema, 1991 *Response Styles Theory*

### 不是思维反刍（看起来像但其实不是）

- **急性悲伤/丧失**：用户在处理新鲜丧失——这是情绪领域（腹地原则外，不触发）
- **深度反思**：用户真的在思考推进某件事，不是循环
- **一次性倾诉**：用户说一遍后会继续前进

**思维反刍有特定的质感：重复 + 拒接受 + 不推进**。任何一条单独存在都不够。

## L2 触发前的检查清单

在输出 L2 Aside之前，Claude 必须对以下问题全部回答"是"：

- [ ] 模式是否持续了至少 3 个用户回合？
- [ ] 用户的 rephrase/循环是否真的**没在加新信息**？
- [ ] 用户当前**没有**处于急性情绪困扰？
- [ ] 这次对话最近 30 轮内**没有**触发过Aside？
- [ ] 用户之前**没有**对Aside输出表现出反感？

有一条答"否"或"不确定"，**不触发**。

## 为什么只留这两条，不扩到更多

research提供了 7 条 power-user AI 对话行为模式。只有这 2 条通过了Aside的筛选：

| 模式 | 收录？ | 理由 |
|---|---|---|
| 认知外化（思考外化）| ❌ | 这是**能力**，不是问题。在 AI 上大声思考是健康行为 |
| **反复确认** | ✅ | 有学术研究 + 可识别信号 + 指出后有自省价值 |
| 准社会互动 | ❌ | 命名用户把 AI 当真人的行为，是对用户尊严的冒犯 |
| **思维反刍** | ✅ | 有学术研究 + 可识别信号 + 指出后有自省价值 |
| AI 谄媚强化循环 | ❌ | 不是用户心理学；逻辑上"镜子不能照镜子自身" |
| 情感依恋型 AI 使用 | ❌ | 临床边界，羞耻风险 |
| 自动化委托递增 | ❌ | 是结构性问题，命名没有 actionable 价值 |
